PEOPLE

ENBIT is a group of academics across Europe with contribution from Austria, Belgium, France, the Netherlands, Sweden, Switzerland and the UK.

DATA SHARING

The primary goal of ENBIT is sharing. Sharing of data under a secured access but also sharing of ideas, protocols and tools.

ENBIT POLICY

To gain access to the repository, you have to register giving your name, full professional address and email. The management team will verify you are a real person belonging to academia or industry, and if so access will be granted.

As user, you have access to all data that are older than a year. Note that depending on datasets, restriction on commercial use may also apply (this is set in your profile at registration). As data depositor, you have access to all of the data.

 

ENBIT SECURITY

Data are stored on secured servers hidden behind firewalls, making it difficult to access during an external cyber-attack. From the ENBIT website front end, every data access is logged and monitored.

 

ANONYMIZATION

Imaging and metadata data are fully anonymized using dedicated software which removes all identifiable tags from images (no personal information). In addition, defacing algorithms that remove face, ear and dental information are used on MRI data so no one can be identified.

Participants’  metadata only contain clinical scores and/or surgical outcomes. Metadata may contain age but not date of birth, country of origin or residence but not address or postcode.

DATA STRUCTURE

The ENBIT repository hosts data organized using the Brain Imaging Data Structure (BIDS). We have our own validator and working on meta-standards to incorporate into the validator.

TOOLS

ENBIT is interested in developping brain imaging of tumours, and along with our data repository, we have a series of tools that can be used to analyze these images - these can be found via the Neuroimaging Informatics Tools and Resources Clearinghouse (NITRC).
